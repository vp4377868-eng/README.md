# Multifunctional AI Assistant (Offline)

## Problem
Internet is not always available. Students and common people need an AI that works offline.

## Solution
This project is an offline AI assistant that can:
- Answer basic questions
- Work without internet
- Support local language
- Help in emergencies

## Features
- Offline text-based AI
- Simple UI
- Emergency help module

## Tech Stack
- Python
- Local datasets
- Android (optional)

## How to Run
1. Download the code
2. Run main.py
3. Use the assistant offline
